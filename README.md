# omnifleet-site

Marketing landing page for [Omnifleet](https://github.com/Neoplanetz) — a multi-robot
fleet control dashboard for ROS 2. Apple-style dark single page, Korean (`/`) and
English (`/en/`), deployed on Cloudflare Pages.

## Develop

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # i18n parity check + astro build → dist/
npm run preview
```

## Media pipeline

`public/videos/*.{mp4,webm}` + `public/posters/*.jpg` are real dashboard screen
recordings (placeholders generated by `scripts/make-placeholders.sh` until the
capture session). Re-capture: `node scripts/capture.mjs --url <app-url> --duration 60`,
then trim/encode with ffmpeg (see scripts/capture.mjs header comment). Keep every
file under 10 MB (Cloudflare Pages hard limit: 25 MB/file).

## Deploy

Cloudflare Pages, GitHub integration: build `npm run build`, output `dist`,
env `NODE_VERSION=22`.
